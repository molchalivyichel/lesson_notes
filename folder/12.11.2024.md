# Опосредование доступа

Это идея о том, чтобы делать переменные приватными и чтобы их можно было изменять через определенные функции, как например эта: 

```cpp
void editAge(unsigned p_age)
{
    if(p_age > 0 && p_age < 140){
        age = p_age;
    }
}
//in main

person.editAge(20);
```
